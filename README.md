Ireland Woodland Afforestation Map 2023

Project for EGM722: 

An interactive folium map for woodlands across each county in the Republic of Ireland for the year 2023. 
This map allows comparison between broadleaf, conifer and total afforestation by hectare.

Installation:
1. Clone this repository from GitHub @ https://github.com/mcatasneyrowan-ui/ireland-woodland-map
2. Load the conda environment
3. Activate the environment: ireland-woodland-map
4. Launch JupyterLab

Usage:
1. Open woodland_map.ipynb in JupyterLab
2. Run all cells to generate the interactive map and download the .html
3. Code is documented using NumPy style docstrings

Data Sources:
1. counties.geojson - Tailte Éireann via data.gov.ie
2. forestry.csv     - Central Statistics Office Ireland AFA01 Afforestation Area 2023 www.cso.ie
